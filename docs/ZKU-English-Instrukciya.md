## 1. Общее описание

**ЗКУ English** — онлайн-платформа для изучения английского языка студентами ЗКУ им. М. Утемісова. Платформа включает:

- Структурированный курс по уровням (A1 → A1.1 → A2 → B1 → B2 → C1)
- Уроки 6 типов: Reading, Listening, Grammar, Writing, Vocabulary, Test
- Систему геймификации (XP, стрики, достижения, лидерборд)
- Кабинет преподавателя с группами и аналитикой
- Кабинет администратора для управления всей платформой
- AI Writing Coach
- Сертификаты о прохождении уровней

**Адрес входа:** `khamadi.online/english/zku/login`

## 4. Кабинет студента

### 4.1 Дашборд (`/student`)

Главная страница студента показывает:
- **Приветствие** с именем студента
- **Статистика:** XP, дней стрика, пройденных уроков
- **Прогресс до следующего уровня** (XP bar)
- **Быстрые действия:** курс, словарь, placement test, writing coach
- **Навыки:** прогресс по Reading / Listening / Writing / Speaking / Grammar
- **Последняя активность:** последние пройденные уроки

### 4.2 Мой курс (`/student/course`)

Список всех уровней платформы:

| Уровень | Название | Статус |
|---------|----------|--------|
| A1 | Beginner | Открыт согласно `current_level` |
| A1.1 | Elementary | Открыт/Заблокирован |
| A2 | Pre-Intermediate | Открыт/Заблокирован |
| B1 | Intermediate | Открыт/Заблокирован |
| B2 | Upper-Intermediate | Заблокирован |
| C1 | Advanced | Заблокирован |

> **Важно:** Студент видит только свой уровень. Остальные заблокированы.

Внутри уровня отображаются:
- **Завершённые модули** (прогресс ≥ 85%)
- **Текущий модуль** (в процессе)
- **Предстоящие модули** (ещё не начаты)
- **Заблокированные модули** (предыдущий не завершён на 85%)

### 4.3 Модуль (`/student/module/[id]`)

Каждый модуль содержит несколько уроков. Порядок:
1. **Reading** — чтение текста
2. **Listening** — прослушивание диалога
3. **Grammar** — теория + практика
4. **Writing** — письменное задание
5. **Vocabulary** — словарные карточки
6. **Final Test** — итоговый тест (🔒 разблокируется после 85% уроков)

**Правила блокировки модулей:**
- Следующий модуль открывается когда текущий пройден на **≥ 85%**
- Финальный тест открывается после прохождения **≥ 85% уроков** модуля

### 4.4 Урок (`/student/lesson/[id]`)

#### Тип: Reading

**Шаги:**
1. **Предварительный вопрос** (подготовка к тексту)
2. **Чтение текста** + кнопка "Слушать" (TTS голос)
3. **Диалог** + кнопка воспроизведения
4. **Ключевые фразы** — кликнуть для прослушивания
5. **Задание 1** (True/False вопросы)
6. **Задание 2** (вопросы с выбором ответа)
7. **Завершение** — XP начислен

#### Тип: Listening

**Шаги:**
1. **Подготовка** — словарь ключевых слов
2. **Прослушивание** — аудиоплеер с кнопками ▶ ⏸ ⏹
   - Текущая реплика подсвечивается
   - Можно ставить на паузу
3. **Задание 1** — True/False
4. **Задание 2** — вопросы с множественным выбором

#### Тип: Grammar

**6 шагов Duolingo-стиля:**
1. Теория (объяснение правила на русском)
2. Формула (структура)
3. Паттерны (примеры)
4. Типичные ошибки
5. Ключевое правило
6. Практика (упражнения)

#### Тип: Final Test (Quiz)

- Вопросы трёх типов: **Multiple Choice**, **True/False**, **Fill in the blank**
- Система комбо: правильные ответы подряд дают ×2 и ×3 XP
- После теста: оценка (A/B/C/D), разбор ошибок по категориям
- Кнопка **"Пройти ещё раз"** — сбросить и начать заново

#### Аудио (TTS)

- Кнопка **▶ Слушать** — воспроизведение голосом
- **⏸ Пауза** — остановить
- **⏹ Стоп** — завершить
- Показывает текущую строку диалога

> Для премиального голоса нужен `OPENAI_API_KEY` в настройках сервера.  
> По умолчанию используется голос браузера.

### 4.5 Словарь (`/student/vocab`)

- Список пройденных словарных уроков
- Карточки слов с переводом (EN → RU/KZ)
- Кнопка "Listen" для каждого слова

### 4.6 Лидерборд (`/student/leaderboard`)

- Топ-50 студентов платформы по XP
- Фильтр: Всё время / Месяц / Неделя
- Ваше место выделено синей строкой

### 4.7 Достижения (`/student/achievements`)

Ачивки за:
- Пройденные уроки (1, 5, 10, 25, 50, 100)
- Стрики (3, 7, 14, 30 дней)
- XP пороги (100, 500, 1000, 5000 XP)
- Уровни (первый урок, первый тест)

### 4.8 Сертификаты (`/student/certificates`)

- Сертификат выдаётся когда студент реально прошёл **≥ 80% уроков** уровня
- До этого — статус "В процессе" или "Заблокирован"
- Сертификат можно перевернуть (flip) чтобы увидеть детали
- Кнопка скачивания PDF

### 4.9 Прогресс (`/student/progress`)

- Детальная статистика по навыкам
- История пройденных уроков с датами и баллами

### 4.10 Профиль (`/student/profile`)

**Вкладки:**
1. **Профиль** — имя, email, телефон, факультет (из данных при регистрации)
2. **Безопасность** — смена пароля
3. **Уведомления** — настройки
4. **Статистика** — XP, стрик, уроки, навыки

### 4.11 Placement Test (`/student/placement`)

Тест для определения уровня. Результат показывает рекомендованный уровень.

### 4.12 Writing Coach (`/student/writing-coach`)

AI-ассистент для проверки и улучшения письменных работ.  
Использует Claude AI для анализа текста.

## 7. Система уровней и блокировки

### 7.1 Уровни студента

Каждый студент имеет `current_level` (текущий уровень). Видит только свой уровень, остальные заблокированы.

**Порядок уровней:**
```
A1 → A1.1 → A2 → B1 → B2 → C1
```

### 7.2 Блокировка модулей

- **Модуль 1** — всегда открыт
- **Модуль N+1** — открывается когда Модуль N пройден на **≥ 85%**
- Заблокированные модули показываются серым с замком 🔒
- Подсказка: "Пройдите М1 на 85% → откроется этот модуль"

### 7.3 Блокировка финального теста

Внутри каждого модуля финальный тест заблокирован пока студент не прошёл **≥ 85% всех уроков модуля** (Reading + Listening + Grammar + Writing + Vocabulary).

Подсказка показывает: `"5 из 6 уроков · нужно ещё 1"`

### 7.4 Сертификаты

Сертификат уровня доступен когда студент прошёл **≥ 80% всех уроков уровня** на платформе.

## 8. XP и геймификация

### 8.1 Начисление XP

| Тип урока | XP за урок |
|-----------|-----------|
| Reading | 60–80 XP |
| Listening | 55–70 XP |
| Grammar | 65–75 XP |
| Writing | 70–90 XP |
| Vocabulary | 35–50 XP |
| Final Test | 100–150 XP |

### 8.2 Комбо в тестах

В финальном тесте при правильных ответах подряд:
- 3 подряд → **×2 XP**
- 5 подряд → **×3 XP**

### 8.3 Стрик

- Увеличивается на 1 каждый день когда студент проходит хотя бы один урок
- Сбрасывается при пропуске дня
- Рекорд стрика сохраняется

### 8.4 Лидерборд

Топ-50 студентов по общему XP. Обновляется в реальном времени.

## 9. Технические детали

| Компонент | Технология |
|-----------|-----------|
| Фронтенд | Next.js 14, TypeScript, Tailwind |

### 9.2 Основные таблицы БД

| `english_lesson_progress` | Прогресс по урокам (score, xp_earned, completed) |

### 9.3 API роуты

| Роут | Описание |
|------|----------|

Новые студенты добавляются через скрипт `scripts/seed-students.mjs`:
```
node scripts/seed-students.mjs
```
Или вручную через Supabase Dashboard → Authentication → Users.

### 9.5 Добавление преподавателя

Через кабинет администратора: **Преподаватели → + Создать преподавателя**.  
Или через скрипт `scripts/seed-mentors.mjs`.

### 9.6 Проблемы и решения

| Проблема | Решение |
|----------|---------|
| Студент не может войти | Сбросить пароль через кабинет Admin → Студенты → 🔑 |
| Группа не отображается | Проверить SQL миграцию 052 в Supabase |
