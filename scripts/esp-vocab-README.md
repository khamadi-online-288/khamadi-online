# ESP Vocabulary SQL Files

Run these files in order in the **Supabase SQL Editor**.

## Step 1 — Cleanup (run first, once)
```
esp-vocabulary.sql          — cleanup block + Computer Science B1
```

## Step 2 — Computer Science B1-C1 (inactive course, optional)
```
esp-vocab-cs-b1c1.sql
```

## Step 3 — Active courses (run all, required)
```
esp-vocab-accounting-b1.sql
esp-vocab-finance-b1.sql
esp-vocab-hospitality-a2.sql
esp-vocab-law-b1.sql
esp-vocab-management-b1.sql
```
